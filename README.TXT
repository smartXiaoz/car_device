NV1.14
	

	
	释放前请更新 information.c 中的版本号，	




MCU上电启动，初始化引脚时功放的STANDBY和MUTE引脚同时被拉高，功放芯片在STANDBY引脚被拉高，刚开始工作时，有一段振荡期，此时因MUTE引脚别拉高，功放处于震荡期，强行拖动负载会造成较大的电流波动，此波动超出功放电源芯片的压力承受范围，引起芯片烧毁。新版程序，在STANDBY拉高以后做延时等待处理，等功放芯片渡过震荡期，稳定工作以后，再拉高MUTE引脚，拖动负载正常工作，避免因不稳定工作造成的电流波动，防止功放电源芯片被烧毁。
